
1. this is not some fancy C++. This is _simple_ C++.
2. avoid defines 
3. avoid globals
4. no exceptions
5. don't use C++11 (C++0x is ok)
6. use custom allocation functions (Alloc/Realloc/Free/New/Delete/NewArr/DeleteArr)
6. why we have typedefs over STL containers? to support cross dll allocation (use Alloc/Realloc/Free/New/Delete/NewArr/DeleteArr for the same reason)
7. bin/libs folders format: <BUILD_SYSTEM>/<PLATFORM>/<CONFIGURATION>. Example 'VS2013/Win32/Release' or 'GCC/Linux/Debug'
9. all functions must use Status to set status (including functions that return StatusCode)
10. all string used are UTF8 (even on windows)
11. tabs for identing (start of the line), spaces for aligning:
int test(int a,
.........int b)
{
<tab>int c;

<tab>c = testfunc(a,
<tab>.............b);
}
12. max line length = 100
