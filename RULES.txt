
1. this is not some fancy C++. This is _simple_ C++.
2. avoid defines 
3. avoid static globals (use static functions that return addr or refs of static declared vars - 
                         for non scalars values it's a must)
4. no exceptions
5. don't use C++11 (C++0x is ok)
6. why do we have typedefs over STL containers? for flexibility in the future
7. bin/libs folders format: <BUILD_SYSTEM>/<PLATFORM>/<CONFIGURATION>. Example 'VS2013/Win32/Release' or 'GCC/Linux/Debug'
8. all functions that return error codes must return Status
9. all strings used are UTF8 (even on windows)
10. tabs for identing (start of the line), spaces for aligning:
int test(int a,
.........int b)
{
<tab>int c;

<tab>c = testfunc(a,
<tab>.............b);
}
11. max line length = 130
